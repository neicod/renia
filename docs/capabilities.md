

# Capabilities

Capabilities are configuration flags that define tenant behavior.

## Core Capabilities

### PRICING_MODE
- PUBLIC
- GROUP_FEW
- GROUP_MANY
- ACCOUNT

### STORE_MODE
- single
- multi

### I18N_MODE
- domain
- path
- off

### CATALOG_RENDERING_MODE
- isr_shell (default)
- ssr

## Rules
- Capabilities must not change component APIs
- Capabilities influence data fetching and cache policy only
